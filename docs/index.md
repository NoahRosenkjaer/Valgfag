# Incident Response
---

## Beskrivelse

Denne webside udgør den skriftlige aflevering for det selvvalgte fordybelsesområde på PBA i IT-sikkerhed. Projektet handler om **Incident Response**, hvor fokus er på at forstå og afprøve, hvordan man reagerer på og analyserer en sikkerhedshændelse i praksis, med eget lab.

Emnet er derfor Incident Response med fokus på SOC analyst L1.
